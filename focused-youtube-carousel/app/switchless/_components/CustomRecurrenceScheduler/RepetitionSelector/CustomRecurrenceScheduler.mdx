## Implementation Details

The `CustomRecurrenceScheduler` component is built using MUI Joy components and includes the following features:

1. Repeat frequency selector (day, week, month, year)
2. Repeat interval input
3. Day of week selector for weekly recurrence
4. End date options (never, on a specific date, or after a number of occurrences)
5. Responsive layout

The component handles various recurrence patterns and provides a user-friendly interface for setting complex schedules.s
